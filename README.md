So this is KETCHUP based on a ketchup bottle but with a little twist. So I like ketchup and with the pcb board I was inspired to use it in the project. With this I made the pcb base look like a ketchup bottle. The LEDs on it form a cap at the top then will blink in a little wave loop under it. It starts at ketchup, then loop, then the word finish where the last LED is.
